CRS Ontology

In this repository we specify vocabularies to describe coordinate and spatial reference systems and their components.
The vocabularies will be more fleshed out as time goes by and are until a release to be considered work in progress.
Current anticipated vocabularies are:
  • Alignments: Alignments of the vocabulary to other vocabularies representing CRS
  • Application vocabulary: Capturing typical applications of defined SRS
  • Core Ontology: Representing core parts for the specification of a CRS
  • Coordinate System Vocabulary: Representing coordinate system types and their parameters
  • Coordinate Operation Vocabulary: Representing operations on coordinates
  • Datum Vocabulary: Representing datum types and their parameters
  • Projection Vocabulary: Representing common projections and their parameters
  • Planet Vocabulary: Representing planets and their approximations e.g. as Ellipsoids
