CRS Ontology

In this repository we specify vocabularies to describe coordinate and spatial reference systems and their components.
The vocabularies will be more fleshed out as time goes by and are until a release to be considered work in progress.
Current anticipated vocabularies are:
  • Alignments: Alignments of the vocabulary to other vocabularies representing CRS
  • Application Module: Capturing typical applications of defined SRS
  • Core Ontology: Representing core parts for the specification of a CRS
  • Coordinate System Module: Representing coordinate system types and their parameters
  • Coordinate Operation Module: Representing operations on coordinates
  • Datum Module: Representing datum types and their parameters
  • Projection Module: Representing common projections and their parameters
  • Planet Module: Representing planets and their approximations e.g. as Ellipsoids
